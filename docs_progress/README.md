# 项目进展文档目录

此目录用于存储项目开发过程中的进展记录、状态更新和里程碑文档。

## 文档类型

- **进展报告** — 记录阶段性开发成果和进度
- **决策记录** — 重要的架构和技术决策
- **问题追踪** — 遇到的问题及解决方案
- **会议纪要** — 项目相关的讨论记录
- **状态更新** — 版本发布、功能完成等状态变化

## 命名规范

- 日期前缀：`YYYY-MM-DD_文档名称.md`
- 示例：`2026-04-14_会话记忆系统实现总结.md`
