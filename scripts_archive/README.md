# 归档脚本目录

此目录用于存储已完成特定任务、临时调试或暂时不使用的脚本。

## 归档原则

- **一次性任务脚本** — 已完成特定任务，不再需要运行
- **临时调试脚本** - 用于调试特定问题，问题已解决
- **验证脚本** — 用于验证某次升级或功能，验证完成后归档
- **修复脚本** — 用于修复特定问题的临时脚本

## 归档清单

### 验证与检查脚本
- `check_modules.py` — 检查模块注册状态
- `verify_upgrade.py` — 验证系统升级结果

### 修复脚本
- `fix_tsr_ranker.py` — 修复 TSR Ranker 模块的临时脚本

---

**注意**：当前活跃的脚本位于：
- 根目录：`run_web.py`（Web 服务启动）、`web_api.py`（主 API）
- `scripts/` 目录：常用工具脚本（如 `build_pdf.py`）
- `testscript/` 目录：所有测试脚本
