# 项目内容文档目录

此目录用于存储项目的技术文档、用户指南和参考资料。

## 文档类型

- **架构设计** — 系统架构、模块设计、数据流图
- **技术文档** — API 文档、配置说明、开发指南
- **用户手册** — 安装指南、使用教程、常见问题
- **参考资料** — 相关论文、技术标准、外部文档链接
- **配置模板** — 配置文件示例、环境变量说明

## 文档结构

```
docs_content/
├── architecture/          # 架构设计文档
├── api/                   # API 文档
├── guides/                # 使用指南
├── references/            # 参考资料
└── templates/             # 配置模板
```

## 命名规范

- 使用英文命名，简洁明了
- 架构文档：`architecture_模块名.md`
- API 文档：`api_端点名称.md`
- 用户指南：`guide_功能名称.md`
